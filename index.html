#1 Coding

Minggu ini adalah minggu pertama saya belajar "coding". I'm just starting to be a professional web developer. Saya mendaftar di sebuah "coding bootcamp" Hacktiv8 namanya. Sebenarnya sudah dari tahun lalu berkeinginan untuk belajar di Hacktiv8 sekaligus pindah profesi. hehe

Background saya adalah seorang lab technician di Puskesmas. Saya lebih banyak aktif dalam program pembernantasa malaria di Maluku. Sama sekali tidak memiliki background IT namun sebenarnya dari sejak "esde" saya sudah mengagumi komputer dan bercita-cita menulis kode program. Saya berpikir dengan background saya sebagai tenaga kesehatan mungkin saya bisa membuat suatu aplikasi baik Web apps maupun Mobile apps yang bisa membantu meningkatkan akses kesehatan bagi masyarakat kita. Atau bisa membantu dalam program pemberantasan malaria di daerah saya dengan hal baru yang saya pelajari sekarang (Computer science).

Coding bootcamp ini sendiri akan saya ikuti hingga november nanti. Saya percaya bahwa menuntut ilmu harus dikerjakan hingga akhir hayat. Dan mempelajari suatu hal baru sangatlah menyenangkan dan sebenarnya selalu sulit di awal tapi waktu akan menjawab. You will always sucks at first but then if you keep trying it will be better.

Kesan saya di minggu pertama di Phase 0 hacktiv8 adalah sungguh berat. Hehe.. Bayangkan tiap hari ada tugas. Dan banyak referensi yang harus di baca serta banyak video tutorial yang harus ditonton di youtube. Git & Github jadi hal paling menguras otak saya, maklum newbie. Dan sementara itu saya masih banyak kerjaan di kantor. Tapi akhirnya anda bisa mengunjungi website pertama saya ini berkat pembelajaran di minggu pertama. Well I think that's all for today. See you next.
